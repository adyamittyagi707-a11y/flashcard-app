# Flashcard Quiz App

This is a simple frontend web application that allows users to create and study flashcards.

## Features
- View questions and reveal answers
- Navigate between flashcards (Next/Previous)
- Add new flashcards
- Edit existing flashcards
- Delete flashcards

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open the app in a browser
2. Add your own questions and answers
3. Use buttons to navigate and study
